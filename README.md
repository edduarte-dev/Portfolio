# Portfólio Pessoal – Eduardo Duarte Cunha

Este repositório contém meu **portfólio pessoal**, desenvolvido como parte do meu aprendizado em **desenvolvimento web**.  
O projeto apresenta uma página inicial com informações sobre mim e uma página de contato.

O foco está na aplicação de **HTML semântico**, **CSS organizado** e boas práticas de layout.

---

## 📌 Funcionalidades

- Página Home com apresentação pessoal
- Página Contato com formulário e mapa incorporado
- Navegação simples entre páginas
- Layout utilizando Flexbox e CSS Grid
- Paleta de cores definida com variáveis CSS

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  

---

## 🚀 Como executar

Abra o arquivo `index.html` no navegador  
*(ou utilize uma extensão como Live Server no VS Code)*
